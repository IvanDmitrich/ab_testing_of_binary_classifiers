# A/B‑тестирование: оценка эффективности ML‑модели
## Краткое описание
Репозиторий содержит исследование по A/B‑тестированию, проведенное в рамках реальной рабочей задачи в ИТ-компании. Цель - оценить целесообразность внедрения ML-модели для автоматизированной проверки релевантности поисковой выдачи.

## В репозитарии представлены
- Сырые дынные оценок асессоров и предсказаний модели;
- Код предобработки, анализа и тестирования;
- Отчет с выводами и рекомендациями.

## Инструменты анализа

1. Обработка и трансформация данных:
   - pandas - загрузка, очистка, агрегация данных;
   - numpy - численные операции и векторизация.
     
2. Статистический анализ:
   - Бутстрап - построение процентильного ДИ для проверки гипотез;
   - Метод Монте-Карло - статистическое планирование эксперимента;
   - statsmodels - тесты стационарности временных рядов ADF/KPSS, линейная регрессия OLS, доверительные интервалы для пропорций;
   - scipy - интерполяция данных через interp1d;
   - sklearn.model_selection - кросс валидация временных рядов (TimeSeriesSplit) для теста на стабильность.
     
3. Оптимизация производительности:
   - numba - ускорение Бутстрапа;
   - joblib - параллелизация итераций метода Монте-Карло.
     
4. Визуализация:
   - matplotlib - построение графиков распределения ошибок, динамики метрик.

## Структура репозитория
